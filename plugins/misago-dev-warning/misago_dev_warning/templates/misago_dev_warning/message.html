Misago code checked out from the misago GitHub repository may be a work in progress and should not be used to run live forums.

Features may be incomplete, unstable, or not final, and there may be no way to upgrade the database to future versions.

To run a live forum, use one of the provided Docker images:

  • Misago Docker (official)
  • Misago Image (community)

To evaluate a released version on your machine, clone one of the release tags from GitHub.
